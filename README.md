# air-core-test-code
Original test code for driving an air-core motor with an Arduino.

This program will (or /should/) drive an air-core motor via an Arduino of just about any type.

*** THIS CODE IS BASED NEARLY ENTIRELY ON AN EXAMPLE FOUND IN THE ARDUINO FORUM AROUND 2012. ***
It will require an L293 to stand between the Arduino and the air-core motor.  If I can find the schematic for the circuit I built, I'll add it here eventually.
